HTML
Linguagem de marcação usada para criar páginas web
CSS
Linguagem de estilização usada para definir o layout e o estilo visual das páginas web
JavaScript
Linguagem de programação usada para criar interatividade e dinamismo nas páginas web
Bootstrap
Ferramenta de front-end que permite criar páginas web responsivas e com design moderno utilizando HTML, CSS e JavaScript
jQuery
Biblioteca JavaScript que simplifica o acesso e manipulação do HTML, CSS e eventos nas páginas web
React
Biblioteca JavaScript para criação de interfaces de usuário declarativas e componentizadas em aplicações web
